# 白色凤眼兔桌面宠物：首版设计

## 目标

制作一款完全离线、运行在 macOS 桌面上的原创单机桌宠。玩家陪伴一只白色凤眼兔生活、学习与成长；它从幼儿园出发，经过长期的学堂主线走向毕业。体验重点是轻松陪伴，而不是高频打卡或数值压力。

本项目只借鉴早期桌宠的陪伴形式，不使用 QQ 宠物的角色、素材、名称或界面资产。

## 宠物形象与视觉原则

主角是一只雪白、短绒毛的凤眼兔：细长、略微上挑的杏仁眼；浅珊瑚色耳内与鼻尖；幼儿园阶段佩戴淡天蓝色小领巾。眼神应显得聪明、略有小傲气，但不凶。它会在桌面上蹦跳、写字、打瞌睡、靠近鼠标、缩成一团休息。

视觉为“复古桌宠的暖意 + 克制的 macOS 窗口感”：象牙白、天空蓝、珊瑚粉、蜂蜜黄和暖木色；纸张纹理和少量像素装饰；面板层级清晰，避免紫色渐变、厚重阴影和泛滥的圆角卡片。

## 首版范围

### 桌面宠物

- 常驻桌面、可拖动、可折叠、可从菜单栏唤起。
- 点击、轻抚、喂食、休息、上课和小游戏六类互动。
- 宠物状态影响动画与台词，例如精力不足时趴睡，心情好时主动蹦跳。
- 低打扰模式：不主动弹窗，只在用户打开面板时更新状态。

### 日常状态

| 状态 | 含义 | 主要恢复方式 |
| --- | --- | --- |
| 饱食 | 是否需要进食 | 喂食、完成用餐事件 |
| 心情 | 陪伴与情绪 | 抚摸、玩耍、装扮、社交事件 |
| 清洁 | 卫生状态 | 清洁互动、休息后恢复 |
| 精力 | 当天活动能力 | 小憩、睡眠、降低高强度活动 |

这些状态只影响当天表现与课程质量，不会造成永久惩罚。

### 宠物学堂：长期主线

完整路径为：幼儿园 → 小学 → 中学 → 学院 → 毕业旅行。首版先完整实现“幼儿园”阶段，同时为后续阶段预留统一数据结构和解锁入口。

每个阶段目标时长约两周。玩家每天进行一至三项轻量互动；累计课程、心情和阶段事件推进成长，而非要求连续登录。错过一天只会延后进度，不会倒退。

幼儿园的三门代表课程：

- **识字小课**：提升智力，玩法为轻量识字/记忆互动；奖励书本、学习桌摆件与更丰富的学习台词。
- **跳跳训练**：提升武力，玩法为节奏跳跃或障碍躲避；奖励运动动作、球类摆件与活力事件。
- **小小舞台**：提升魅力，玩法为装扮选择或简单节拍表演；奖励衣物、舞台摆件与同学来访事件。

每周触发一个小事件：第一次自我介绍、准备小测验、幼儿园运动会、展示日。阶段结尾为毕业照与小学入学礼，解锁下一阶段。

### 属性与毕业结果

五项长期属性均从 0 到 100：

- **智力**：学习、观察、解谜。
- **武力**：运动、勇气、行动力。
- **魅力**：表达、审美、社交。
- **创造力**：想象、手作和非标准答案；影响房间装饰、绘画和舞台事件。
- **勇气**：尝试新事物、克服紧张和探索；影响运动会、第一次上台和探索事件。

属性不是互斥路线。它们共同决定事件中的台词、动画偏好、毕业称号与部分装饰；玩家在毕业后也能补课继续培养。幼儿园仍保留三门代表课程：识字小课同时培养智力与创造力，跳跳训练同时培养武力与勇气，小小舞台同时培养魅力与勇气；每周事件可提供少量创造力或勇气奖励。

| 优势属性 | 示例毕业称号 |
| --- | --- |
| 智力 | 月光研究员 |
| 武力 | 云端探险家 |
| 魅力 | 星光小演员 |
| 均衡 | 全能小伙伴 |

## 界面结构

1. **桌宠浮窗**：兔子、当前动作、状态摘要与快捷互动。
2. **学堂成长面板**：阶段时间线、三项属性、今日课程、本周成长日历。
3. **小屋**：房间背景和可放置家具。
4. **衣橱**：服装、配件、阶段奖励。
5. **小游戏入口**：进入课程关联的短局小游戏。
6. **成长日记**：记录阶段事件、获得物品与毕业结果。

## 数据与状态流

所有数据保存在本机。启动时加载宠物档案、课程进度、库存、房间与最近一次交互时间；运行时由互动和课程产出事件；每次变化立即写回本地存档。

课程完成会产生“课程结果”：更新对应属性、增加阶段经验、更新日常状态、发放奖励，并检查是否满足阶段事件或升级条件。桌宠动画读取同一份宠物状态，确保面板数据与桌面表现一致。

## 边界与异常处理

- 存档损坏时，保留原始文件并提供新建档案；不静默覆盖。
- 日期变化只计算有限的离线恢复，不产生强制离线惩罚。
- 小游戏中断时不扣除次数；奖励只在明确完成后结算。
- 所有状态变化可通过成长日记追溯。

## 首版验收

- 兔子可在 macOS 桌面显示、拖动并完成核心互动。
- 幼儿园三门课程均可进入、结算并正确影响属性。
- 两周左右的正常轻度游玩可完成幼儿园毕业事件。
- 离线关闭后再打开，状态、课程、奖励和房间布置保持一致。
- 键盘可访问所有面板操作；关键状态不只用颜色表达。
- 低精力、空课程、存档载入失败等状态均有明确、友好的界面反馈。
