# 课程入口清晰度设计

## 目标

保留“今日课程”标题，让用户在首次看到学堂时能立刻知道三张彩色课程卡就是上课入口。

## 改动

- 三张课程卡显示完整课程名称和简短属性收益，而非只依赖图标。
- 卡片保留现有颜色，但增加“点击上课”可见提示和明确的按钮可点击状态。
- 点击课程后在课程区显示一次简短结果文本，例如“识字小课完成：智力 +8，获得 1 枚学习印记”。
- 成长事件卡保留领取功能，但采用更轻的浅色信息样式，避免与课程入口竞争。
- 不新增计时器、循环动画或后台任务；结果文本只由用户点击更新。

## 验收

- 首屏三门课程都能看见完整中文名称。
- 用户无需猜测即可识别上课入口。
- 点击任一课程能看到对应的文字反馈。
- 既有课程、成长事件与印记结算不变；完整测试通过。
