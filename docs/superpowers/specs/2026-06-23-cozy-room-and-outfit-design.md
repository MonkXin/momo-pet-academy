# 温馨小屋与装扮设计

## 目标

把既有课程、阶段事件和周成长奖励转化为可见、可摆放、可佩戴的长期成果。首版做“温馨卧室 + 学习角”混合空间，不引入自由拖拽、抽卡、商店或常驻渲染。

## 小屋结构

- 休息区：小床、软地毯、窗边盆栽。
- 学习角：书桌、书架、奖牌墙、展示台。
- 家具采用固定位置槽位；每件奖励只有未摆放和已摆放两种状态。
- 小屋只在用户打开 sheet 时创建；关闭后不保留额外渲染负担。

## 奖励映射

| 奖励 | 类型 | 表现 |
| --- | --- | --- |
| 蓝色领结、小红领巾 | 配饰 | 小白颈部 |
| 认真小贴纸、班级小贴纸 | 纪念 | 学习角贴纸墙 |
| 云朵绘本 | 家具 | 书架绘本 |
| 星星奖牌 | 家具 | 奖牌墙 |
| 小舞台摆件 | 家具 | 展示台 |
| 彩虹积木、小阅读灯 | 家具 | 地毯旁或书桌角 |
| 小明星徽章、校园小明星别针 | 配饰 | 小白领结或领巾附近 |

## 交互

- 保留现有“小屋”入口；默认展示已摆放家具。
- 小屋下方按可摆放与已摆放显示奖励，点击切换并立即保存。
- 衣橱仍负责装备配饰；学堂头像与桌宠读取同一当前配饰。
- 首次放入新家具只显示一次静态文字反馈。

## 兼容、测试和性能

- 现有 `rewards`、`placedFurniture`、`equippedAccessory` 继续作为唯一存档来源；旧档无需迁移。
- 测试覆盖摆放、收起、空状态、映射、旧档兼容与家具/配饰互不影响。
- 不新增后台任务、循环动画、WebView 或桌宠常驻家具图层。
