# 奶茶手动启动 App 打包设计

## 目标

提供一个可双击运行的 `奶茶的学堂时光.app`，让用户无需终端即可手动打开奶茶；应用启动后默认显示桌宠。

## 范围

- 基于现有 release 二进制创建标准 macOS `.app` 目录。
- 生成 `Info.plist`、复制可执行文件和 SwiftPM 资源包。
- 使用奶茶现有形象生成标准多尺寸 `icns` 应用图标。
- 输出到 `dist/奶茶的学堂时光.app`。
- 提供可重复运行的本地打包脚本与简短使用说明。

## 不做

- 不注册开机启动或登录项。
- 不创建后台守护进程。
- 不进行 Apple 开发者签名、公证或公开分发。

## 验收

- Finder 双击 `.app` 能直接出现奶茶桌宠。
- Dock/Finder 显示奶茶图标和应用名。
- 退出应用后没有残留进程。
- release 构建与完整测试通过；桌宠静置继续满足低资源基线。
